## 📞 Support Information

### Quick Troubleshooting Guide

**Q: Images still not showing?**  
A: Check API returns imageUrl/imageAsset. Verify URLs are accessible.

**Q: Language buttons not responding?**  
A: Clear browser cache. Refresh page. Check console for errors.

**Q: Scan count stuck?**  
A: Clear localStorage. Verify device ID in cookies.

**Q: Audio not playing?**  
A: Ensure browser supports Web Speech API. Check audio permissions.
